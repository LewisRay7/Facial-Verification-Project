# ExamVerify Flutter Client

Cross-platform Flutter client for the ExamVerify biometric exam authentication system.

## Status

- Windows and Android project folders are scaffolded.
- `lib/main.dart` contains the first responsive ExamVerify dashboard shell.
- Backend/API integration is pending.

## Local Checks

```powershell
flutter analyze
flutter test
```

## Remaining Tooling

- Windows desktop builds need Visual Studio with the Desktop development with C++ workload.
- Android builds need Android Studio and the Android SDK.
